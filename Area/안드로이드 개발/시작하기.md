# 1. 안드로이드 스튜디오 설치
```cmd
winget install Google.AndroidStudio
```

- 앱을 개발하기 위해서 안드로이드 SDK 및 몇몇 도구들이 필요 
  → 안드로이드 스튜디오가 제공

# 2. 프로젝트 생성
- SDK (Software Development Kit) 설정 
	- Minimum SDK : API 24 (Android 7.0)

		- AndroidManifest.xml (기본 설정 파일)
